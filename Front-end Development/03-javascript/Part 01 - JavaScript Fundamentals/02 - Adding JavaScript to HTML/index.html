Adding JavaScript to HTML

Learning three ways to add JavaScript to a webpage.

1. Inline JavaScript

Inline JavaScript is written directly inside an HTML element.

2. Internal JavaScript

Internal JavaScript is written inside a <script> element in the HTML file.

Open the browser console to see the internal JavaScript message.

3. External JavaScript

External JavaScript is written in a separate file such as script.js.

This is the recommended method for keeping larger projects clean and organized.

Open the browser console to see the external JavaScript messages.

Which Method Should I Use?

Inline

Useful for very small examples, but it can make HTML difficult to read.

Internal

Useful for small webpages that only need a little JavaScript.

External

Recommended because it keeps HTML and JavaScript in separate files.
