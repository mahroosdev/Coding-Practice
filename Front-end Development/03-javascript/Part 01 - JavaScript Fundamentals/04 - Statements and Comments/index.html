JavaScript Statements and Comments

Learning how JavaScript instructions and comments work.

JavaScript Statements

A JavaScript statement is an instruction that tells the browser to perform an action.

console.log("This is a JavaScript statement.");

JavaScript normally reads and runs statements from top to bottom.

Single-Line Comments

A single-line comment begins with two forward slashes.

// This is a single-line comment

JavaScript ignores comments when it runs the program.

Multi-Line Comments

A multi-line comment can cover more than one line.

/*
  This is a multi-line comment.
  It can continue on another line.
*/
