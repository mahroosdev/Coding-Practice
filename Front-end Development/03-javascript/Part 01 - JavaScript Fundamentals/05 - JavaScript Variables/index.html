What is a Variable?

A variable is a named container used to store a value.

let studentName = "Mahroos";

The variable name is studentName, and the stored value is "Mahroos".

Why Use Variables?

Store Values

Variables can store information that a program needs.

Reuse Values

A stored value can be used again by writing its variable name.

Change Values

Some variable values can be updated while the program is running.
