JavaScript Variable Keywords

JavaScript provides three keywords for creating variables: let, const, and var.

let learningStatus = "In Progress";
const courseName = "JavaScript";
var studentLevel = "Beginner";

When Should I Use Them?

Use const

Use const when the variable should not receive a completely new value.

Use let

Use let when the variable value needs to change later.

Avoid var for New Code

var is an older keyword. It may still appear in older JavaScript projects.
