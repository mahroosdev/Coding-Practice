What Is an Expression?

An expression is code that produces a value.

What Is a Statement?

A statement is a complete instruction that tells JavaScript to perform an action.

JavaScript Execution Order

JavaScript normally reads and runs statements from top to bottom.

const firstNumber = 10; const secondNumber = 5; const answer = firstNumber + secondNumber; console.log(answer);

Semicolons

A semicolon can mark the end of a JavaScript statement.

Recommended

const language = "JavaScript";

The statement ends clearly with a semicolon.

Possible

const language = "JavaScript"

JavaScript may add the semicolon automatically.
