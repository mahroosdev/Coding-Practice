JavaScript String Methods

Learning how JavaScript reads, searches, cleans, extracts, and changes string values.

What Is a String Method?

A string method is a built-in JavaScript action that can be used with a string.

courseName.toUpperCase()

Most string methods use parentheses after the method name.
