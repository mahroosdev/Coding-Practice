Condition Order

JavaScript stops after the first true condition. Write more specific or higher conditions before broader conditions.

First

score >= 90

Second

score >= 75

Third

score >= 50

Final

else

Else If Summary

Part Purpose
if Checks the first condition
else if Checks another condition
else Runs when no condition matches
Condition order Checks from top to bottom
