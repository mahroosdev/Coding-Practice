Function Parameters and Return Values

Learning how functions receive values and return results.

What Is a Parameter?

A parameter is a variable written inside a function's parentheses.

function greetUser(name) { console.log("Hello " + name); }

1. Arguments

An argument is the value passed to a function when it is called.

2. Multiple Parameters

A function can receive more than one value.

3. Return Values

The return keyword sends a result back from a function.
