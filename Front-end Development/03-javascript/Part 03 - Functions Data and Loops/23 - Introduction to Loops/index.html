What Is a Loop?

A loop repeats a block of code while a condition allows it.

for (let number = 1; number <= 5; number++) { console.log(number); }

3. Main Parts of a Loop

Starting Value

let number = 1

Creates the loop counter.

Condition

number <= 5

Controls whether the loop continues.

Update

number++

Changes the counter after each repetition.

4. Main Loop Types

for Loop

Useful when the number of repetitions is known.

while Loop

Repeats while a condition remains true.

do while Loop

Runs once before checking the condition.

6. Avoiding Infinite Loops

A loop must eventually reach a false condition.

Always update the loop counter or condition. Without an update, the loop may continue forever.

Loop Summary

Concept Purpose
Loop Repeats a code block
Counter Tracks the current repetition
Condition Controls when the loop stops
Update Changes the counter
Infinite loop A loop that never reaches its stopping point
