JavaScript For Loop

Learning how to repeat JavaScript code a known number of times.

What Is a For Loop?

A for loop repeats a block of code while its condition remains true.

for (let number = 1; number <= 5; number++) { console.log(number); }

1. Parts of a For Loop

Starting Value

let number = 1

Creates the loop counter.

Condition

number <= 5

Controls how long the loop runs.

Update

number++

Changes the counter after every repetition.
