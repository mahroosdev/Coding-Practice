While and Do While Loops

Learning how JavaScript repeats code using while and do while loops.

What Is a While Loop?

A while loop repeats code while its condition is true.

let number = 1; while (number <= 5) { console.log(number); number++; }

A do while loop runs its code before checking the condition.

While Compared with Do While

while

Checks the condition before running the code.

do while

Runs the code before checking the condition.

Avoid Infinite Loops

Always update the counter or value used by the condition. Otherwise, the condition may remain true forever.

Loop Summary

Loop Condition Check Minimum Runs
while Before the code block Zero
